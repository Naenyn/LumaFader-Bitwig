Connection

  1. Use Chrome or Edge (Web Serial).
  2. Plug in LumaFader over USB — do not hold Pico BOOT.
  3. Click Connect and choose the LumaFader serial port.
  4. Edit settings, then Save to device. Hard-reset (unplug/replug) to apply.

Stock LumaFader (non-Bitwig) firmware uses the Cherry web config.
