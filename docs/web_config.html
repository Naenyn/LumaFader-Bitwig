Connection

  1. Use Chrome or Edge (Web Serial).
  2. Plug in LumaFader over USB — do not hold Pico BOOT.
  3. Click Connect and choose the LumaFader serial port.
  4. Edit settings, then Save to device. Hard-reset (unplug/replug) to apply.

Stock LumaFader (non-Bitwig) firmware uses the Cherry web config.

MIDI channel & sensitivity

Fader CC numbers

CC numbers the hardware faders send to your DAW (absolute mode). Learn fills a field: click Learn, then send one MIDI CC to the LumaFader USB port (e.g. move a control in Bitwig that targets this device). The editor copies that CC# — it does not map another controller’s fader to the LumaFader physically.

Bitwig action CCs (fixed protocol)

Focus and Four-Track modes talk to the LumaFader Bitwig extension only. When you perform a gesture, firmware sends a short CC pulse on these numbers; the extension listens for the same hardcoded CCs. You configure which button gesture triggers each action below — not the CC numbers.
